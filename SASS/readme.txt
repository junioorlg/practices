 - Para crear archivo de sass a css:
Ejemplo: sass archivo-origen-sass.sass:archivo-destino-css.css
Ejemplo: sass main.sass:main.css
 - Para sobreescribir el archivo .css cada ves que salves el .sass:
Ejemplo: sass --watch archivo-origen-sass.sass:archivo-destino-css.css
Ejemplo: sass --watch main.sass:main.css
